# SMART UPI FRAUD DETECTION USING ADVANCED MACHINE LEARNING ALGORITGHMS:AN INTELLIGENT SYSTEM FOR IDENTIFYING ANAMOLIES IN UPI PAYMENT NETWORKS

# Features:
-> Real-time UPI transaction fraud detection with 96.80% accuracy
-> Compares 3 with ML models: Naive Bayes, Decision Tree, Random Forest
-> Interactive dashboard with fraud visualizations and prediction history
-> CSV upload support with automated preprocessing pipeline
-> User authentication, profile management, and downloadable reports

# Tech Stack:
**ML & Data:** Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
**Backend:** Django, SQLite3
**Frontend:** HTML, CSS, JavaScript
**Tools:** Jupyter Notebook, VS Code, Anaconda
